 Admin Analytics Dashboard

A Flask-based web dashboard for visualizing analytics data.

Features
- Total Users Statistics
- File Upload Analytics
- URL Creation Analytics
- Recent Activities Table
- Interactive Charts

 Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap 5
- Chart.js

 Run Project

pip install -r requirements.txt

python app.py
